# sql-data-analytics-project

A comprehensive collection of SQL scripts for data exploration, analytics, and reporting. This repository is designed to help data analysts, data scientists, and BI professionals quickly analyze relational databases using efficient and reusable SQL patterns.

Each script focuses on a specific analytical theme such as database exploration, KPI calculation, time-based trends, cumulative analysis, segmentation, and more.

---

## 📊 Overview

This repository contains well-structured SQL queries that demonstrate best practices for:

- Data exploration and profiling
- Business metrics and KPIs
- Time-series analysis
- Cohort and segmentation analysis
- Cumulative and rolling calculations
- Reporting-ready queries for BI dashboards
